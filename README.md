# Finalproject-Slither-Speller
 Slither Speller
A terminal-based snake game built in C where the snake 
collects falling words instead of food.

## Technologies Used
- C (standard libraries)
- Dev C++ / VS Code

## How to Run
1. Compile using any C compiler: gcc slither_speller.c -o game
2. Run the executable: ./game

## Features
- Snake collects falling words to score points
- Collision detection
- Score tracking system
